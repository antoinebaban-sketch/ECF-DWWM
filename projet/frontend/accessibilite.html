Déclaration d'accessibilité

Établie le 1er janvier 2026 — Référentiel RGAA 4.1

État de conformité

Le site viteetgourmand.fr est en conformité partielle avec le RGAA 4.1 en raison des non-conformités et des dérogations listées ci-après.

Résultat des tests

Taux de conformité global : environ 75 % des critères RGAA vérifiés

Résultats des tests

Critères conformes

  • Langue de la page définie (lang="fr")
  • Titre de chaque page renseigné
  • Navigation clavier possible sur l'ensemble du site
  • Attributs aria-label sur les éléments interactifs clés
  • Formulaires avec labels associés
  • Messages d'erreur liés aux champs concernés
  • Contrastes de couleurs conformes pour le texte principal
  • Images décoratives avec alt=""
  • Structure de titres hiérarchique (h1 → h2 → h3)
  • Responsive : utilisable sur mobile et tablette

Non-conformités identifiées

  • Certaines images de menus n'ont pas d'alternative textuelle descriptive suffisante
  • Les boutons d'étoiles (notation) pourraient bénéficier d'une meilleure description pour les lecteurs d'écran
  • L'animation CSS de la galerie de photos peut être perturbante pour les utilisateurs sensibles aux mouvements

Dérogations

  • Charge disproportionnée : les documents PDF (devis, factures) ne sont pas entièrement accessibles — une alternative textuelle est proposée sur demande

Technologies utilisées

  • HTML5
  • CSS3 (custom properties)
  • JavaScript (ES2020+, sans framework)

Agents utilisateurs testés : Chrome 120+, Firefox 121+, Safari 17+ (macOS et iOS), Edge 120+, NVDA + Chrome (Windows).

Retour d'information et contact

Si vous constatez un défaut d'accessibilité sur ce site, vous pouvez :

  • Envoyer un email à contact@viteetgourmand.fr
  • Utiliser le formulaire de contact

Nous nous engageons à vous répondre sous 5 jours ouvrés et à traiter votre demande dans un délai de 30 jours.

Voie de recours

Si vous n'obtenez pas de réponse satisfaisante, vous pouvez contacter le Défenseur des droits :

  • Par téléphone : 09 69 39 00 00
